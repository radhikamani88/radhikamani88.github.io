 Fedora hubs

An online collaboration tool for Fedora Linux OS contributors

Fedora is a Linux OS distro that like other linux OS, wants to increase adoption among a non-developer user base. This is only possible if non-developers contribute to improving the user experience of the operating system. A modern online collaboration tool is needed to foster such contributions. I worked on design of this collaboration tool called Fedora hubs
